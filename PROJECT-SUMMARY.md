## 🎨 Quick Customization

### Change Institution Name
- Open `index.html` with Notepad
- Find "Broward-Miami Health Institute"
- Replace with your school name
- Save file

### Change Program Cost
- Open `app.js` with Notepad
- Find line: `baseAmount = 12500`
- Change 12500 to your cost
- Save file

### Add Your Video
- Save your video as: `enrollment-process.mp4`
- Place in same folder as `index.html`
- That's it! It will auto-play

### Change Colors
- Open `styles.css`
- Find `:root {` at top
- Modify color values:
  - `--primary-color: #0066cc;` (main color)
  - `--secondary-color: #00a86b;` (accent)
- Save

### Update Contact Info
- In `index.html` search for "Student Services"
- Update phone numbers and emails
- Save
