## 💾 DATA BACKUP & RECOVERY

### How Data is Stored
- Browser's localStorage (local to each device/browser)
- Persists between sessions
- Survives browser restart

### To Export Data
```javascript
// In browser console (F12)
localStorage.getItem('pn-enrollment-data')
```

### To Clear Data
```javascript
// In browser console (F12)
localStorage.removeItem('pn-enrollment-data')
```

### To Backup
- Copy browser data/storage folder
- Or export JSON data
- Or print/PDF documents
